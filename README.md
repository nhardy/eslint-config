eslint-config-nhardy
--------------------

An ESLint config designed to be similar to `eslint-config-mi9`, which also extends `eslint-config-airbnb`.

Usage
=====
In your root `.eslintrc`:
```json
{
  "extends": "nhardy"
}
```

In your `tests/.eslintrc` file:
```json
{
  "extends": "nhardy/tests"
}
```
